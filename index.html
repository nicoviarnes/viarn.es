Plantify.io

github

Plantify is a houseplant/garden management system. It is integrated with a computer vision API that can recognize over 300,000 different plant species by their scientific names.

Plantify combines that unique ability with a custom web scraper to pull interesting information about the identified plant. Users can then set care schedules, make notes about their plants, and get notified when they need to water or a feed a plant!

Built with Mongo, express, React, Node, Axios, Cheerio

pooblah.io

github

pooblah.io is a tool to analyze the most profitable exchange ratios for currency items in the online game Path of Exile.

The project is built with Angular, MySQL, and Node and is currently running on an AWS LightSail instance
